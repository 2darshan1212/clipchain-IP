**Creator Plan - $29/month:**

**Pro Plan - $99/month:**
- All Creator features
- White-label clips
- API access
- Dedicated support
- **Target:** 30% of paid users

**Average Revenue Per User (ARPU):** $40/month

### 2. Licensing Fees

**How it works:**
- Creators set clip price ($0.05 - $5.00)
- Platform takes 5% of every license
- Remaining 95% split: 70% host, 25% guest

**Example Transaction:**
```
Clip License Price: $2.00
Platform Fee (5%):  $0.10
Host (70% of $1.90): $1.33
Guest (25% of $1.90): $0.475
```

**Projected Licensing Revenue:**
- 10% of clips get licensed (Year 1)
- Average price: $2.00

- Grows to $300K/year by Year 3 (10M clips)

**Pricing:**

## Competitive Analysis

### Direct Competitors

**1. Descript**
- **Strengths:** Established brand, great editor
- **Weaknesses:** No IP registration, no royalties
- **Our advantage:** Blockchain ownership + licensing

**2. Riverside.fm**
- **Strengths:** High-quality recording
- **Weaknesses:** No AI clips, expensive
- **Our advantage:** AI-first, better pricing

**3. Headliner**
- **Strengths:** Social video creation
- **Weaknesses:** No AI, manual process
- **Our advantage:** Automated workflow

### Indirect Competitors

**YouTube:**
- Clips exist but no licensing infrastructure
- We integrate with YouTube as content source

**TikTok/Instagram:**
- Consumption platforms, not creation
- We enable licensed content for these platforms

**Otter.ai:**
- Transcription only, no clips or licensing
- We use transcription as input to our pipeline

### Competitive Moat

1. **First-mover advantage** - Only blockchain IP platform for podcasts
2. **Network effects** - More clips = better discovery
3. **Technology barriers** - AI models, blockchain integration
4. **Data advantage** - Transcript corpus for ML training
5. **Creator lock-in** - IP registered on our platform
